## 功能说明

### 语法格式

```
:::longimage[可选标题]
![图片描述](图片URL|可选尺寸)
:::
```

**或者使用 Obsidian 嵌入语法：**

```
:::longimage[可选标题]
[[图片路径|可选别名]]
:::
```

### 支持功能

1. ✅ **垂直滚动**：长图可上下滚动浏览
2. ✅ **高度限制**：最大高度为视窗的 80%
3. ✅ **标题支持**：可选的标题显示
4. ✅ **平滑滚动**：smooth scroll 效果
5. ✅ **自适应宽度**：图片宽度自动适应容器
6. ✅ **视觉优化**：圆角、阴影、渐变背景
7. ✅ **缩放保持**：保持原始图片宽高比

### 视觉效果

- **布局**：垂直滚动容器
- **最大高度**：80vh（视窗高度的80%）
- **宽度**：100%自适应
- **圆角**：8px圆角边框
- **阴影**：subtle shadow效果
- **滚动**：平滑滚动，scroll-behavior: smooth
- **背景**：淡橙色渐变，与极客杰尼主题匹配

### 使用场景

- 产品架构图
- 系统流程图
- 用户体验地图
- 信息图表
- 长截图
- 思维导图
- 复杂的数据可视化
- 完整的界面截图

### 兼容格式

- JPG/JPEG
- PNG
- GIF
- WebP
- SVG（矢量图）

### 注意事项

1. **图片尺寸**：建议使用长图（宽度≥700px，高度≥1000px）
2. **加载性能**：超大图片可能加载较慢，建议压缩后使用
3. **滚动体验**：可以通过滚动条或鼠标滚轮浏览图片
4. **移动端**：在移动设备上也能正常使用，支持触摸滑动

### 与Gallery的区别

| 功能 | Gallery | LongImage |
|------|---------|-----------|
| **方向** | 横向滚动 | 垂直滚动 |
| **图片数量** | 多张 | 单张 |
| **布局** | 并排展示 | 纵向展示 |
| **适用场景** | 多图对比 | 长图详情 |
| **最大高度** | 不限 | 80vh |

## 高级用法

### 1. 结合代码块使用

```markdown
## 架构设计说明

以下是完整的产品架构设计图：

:::longimage[产品架构图 v2.0]
![完整架构图](https://images.unsplash.com/photo-1551288049-bebda4e38f71?w=800&h=1200)
:::

主要组件说明：
1. 用户界面层
2. 业务逻辑层
3. 数据存储层
```

## 最佳实践

1. **图片优化**
   - 使用压缩后的图片提高加载速度
   - 建议宽度为 800-1000px
   - 高度根据内容调整

2. **标题使用**
   - 使用简洁明了的标题
   - 包含版本号或日期（如：架构图 v2.0）
   - 避免过长的标题

3. **组合使用**
   - 可以与 Gallery 功能组合使用
   - Gallery 展示缩略图，LongImage 展示详细大图
   - 提升用户体验

4. **响应式设计**
   - 在不同屏幕尺寸下都能正常显示
   - 移动端会自动调整滚动方式
   - 保持良好的阅读体验
