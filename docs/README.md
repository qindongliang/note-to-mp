## 💡 使用技巧

### 🌟 高效使用
1. **设置默认主题**：在插件设置中保存常用主题
2. **使用模板**：创建包含前言元数据的模板笔记
3. **快捷键**：使用 `Ctrl+P` 快速打开命令面板
4. **自定义 CSS**：打造独特的排版风格

### 🎨 样式推荐
- **技术博客**：MWeb Panic + Github 高亮
- **日常随笔**：Maple + GitHub 高亮
- **学术文章**：Solarized Light + Atom One Light
- **护眼模式**：Ayu（深色主题）

### 📝 最佳实践
- 定期备份笔记内容
- 图片大小控制在 2000px 以内
- 避免单篇文章过长（建议拆分为多篇）
- 使用有意义的文件名（避免中文和特殊字符）

## ⚠️ 重要提醒

### 安全提示
- 插件仅在本地处理数据，不会上传您的笔记内容
- 图片上传使用微信官方 API，确保数据安全
- 订阅功能仅验证激活状态，不存储个人信息

### 使用限制
- 复制内容大小建议不超过 1MB
- 单次上传图片不超过 10 张
- 图片格式仅支持 JPG/PNG/GIF
- 文件大小不超过 10MB

### 兼容性
- 支持 Obsidian 1.4.5 及以上版本
- 完全兼容 macOS、Windows、Linux
- 移动端需要配合桌面端使用
