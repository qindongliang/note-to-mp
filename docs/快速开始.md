# NoteToMP 快速开始指南

欢迎使用 NoteToMP！本指南将帮助您在 5 分钟内快速上手这个强大的 Obsidian 插件。

## 📦 第一步：安装插件

### 1.1 准备工作
- 确保您已关闭 Obsidian 的安全模式
  - 打开 Obsidian → 设置 → 第三方插件 → 关闭安全模式

### 1.2 安装插件
1. 打开 Obsidian 设置
2. 进入 **第三方插件** → **社区插件市场**
3. 搜索 **NoteToMP**
4. 点击 **安装**

### 1.3 下载主题资源
插件安装完成后，还需要下载主题和代码高亮样式：

**方法一：通过插件设置下载（推荐）**
1. 打开 Obsidian 设置
2. 进入 **第三方插件** → **Note to MP**
3. 找到 **获取更多主题** 部分
4. 点击 **下载** 按钮

**方法二：手动下载**
1. 访问 [GitHub Releases](https://github.com/sunbooshi/note-to-mp/releases)
2. 下载 `assets.zip` 文件
3. 解压到 `.obsidian/plugins/note-to-mp/assets/` 目录

## 🚀 第二步：第一次使用

### 2.1 打开插件
- **方法一**：点击 Obsidian 左侧工具栏中的 NoteToMP 图标
- **方法二**：按 `Ctrl+P`（Mac: `Cmd+P`）打开命令面板，搜索 **复制到公众号**

### 2.2 选择样式
1. 插件界面打开后，在 **样式** 下拉菜单中选择喜欢的主题
2. 目前提供 30+ 款精美主题，持续增加中

### 2.3 复制内容
1. 检查预览效果
2. 点击 **复制** 按钮
3. 到微信公众号编辑器中粘贴（`Ctrl+V` 或 `Cmd+V`）

## 🔧 常用设置

### 设置默认样式
1. 进入插件设置
2. 在 **默认样式** 中选择您喜欢的主题
3. 点击 **保存**

### 自定义 CSS
如果您想自定义样式：
1. 创建一篇新笔记，命名为「自定义样式」
2. 在笔记中写入 CSS 代码，例如：
```css
.note-to-mp {
    font-family: "Microsoft YaHei", sans-serif;
}
```
3. 在插件设置中的 **自定义 CSS 笔记** 字段填入「自定义样式」

## 📱 高级功能（需订阅）

### 多公众号管理
- 在 **公众号** 下拉菜单中切换不同公众号

### 上传图片
- 点击 **上传图片** 将本地图片上传到微信公众号
- 替换预览中的图片地址

### 发送草稿
- 点击 **发草稿** 直接发送到公众号草稿箱
- 无需手动复制粘贴
