## 1. 安装相关

### Q1.1：在社区插件市场找不到 NoteToMP，怎么办？

**A：** 请检查以下几点：
1. 确认 Obsidian 版本为 1.4.5 或更高版本
2. 确认已关闭 Obsidian 的安全模式
   - 设置 → 第三方插件 → 关闭安全模式
3. 尝试重启 Obsidian
4. 检查网络连接，必要时使用科学上网

如果仍有问题，请加微信 `Genius35Plus` 获取帮助。

---

### Q1.2：插件安装成功，但没有看到工具栏图标？

**A：** 图标可能在隐藏的工具栏中。请尝试：
1. 右键点击左侧工具栏
2. 选择「自定义」
3. 拖拽 NoteToMP 图标到可见位置

或者使用快捷键 `Ctrl+P` (Mac: `Cmd+P`)，搜索「复制到公众号」。

---

### Q1.3：主题下载失败，一直显示加载中？

**A：** 可能的原因和解决方案：

**原因 1：网络问题**
- 尝试使用科学上网
- 检查防火墙设置

**原因 2：插件目录权限问题**
```bash
# 确保目录可写
chmod -R 755 ~/.obsidian/plugins/note-to-mp/
```

**原因 3：手动下载**
1. 访问 [GitHub Releases](https://github.com/sunbooshi/note-to-mp/releases)
2. 下载 `assets.zip`
3. 解压到 `.obsidian/plugins/note-to-mp/assets/`

---

### Q1.4：安装后只有默认主题，其他主题显示空白？

**A：** 这是主题资源未正确下载的表现。请按以下步骤操作：

1. **检查目录结构**
```
.obsidian/plugins/note-to-mp/
├── assets/
│   ├── themes/         # 应该有 30+ 个 CSS 文件
│   └── highlights/     # 应该有 20+ 个 CSS 文件
├── main.js
└── manifest.json
```

2. **重新下载主题资源**
   - 进入插件设置
   - 点击「获取更多主题」→「下载」
   - 等待下载完成

3. **如果下载失败，手动操作**
   - 下载 assets.zip
   - 解压到正确位置
   - 重启 Obsidian

---

### Q1.5：插件更新后功能异常？

**A：** 请尝试：
1. 完全重启 Obsidian
2. 禁用插件后再重新启用
3. 如果问题持续，清理插件数据：
   - 关闭 Obsidian
   - 删除 `.obsidian/plugins/note-to-mp/data.json`
   - 重新启动 Obsidian

## 2. 使用相关

### Q2.1：复制到公众号后样式丢失？

**A：** 可能原因：

**原因 1：剪贴板权限**
- 检查系统是否允许 Obsidian 访问剪贴板
- macOS：系统偏好设置 → 安全性与隐私 → 隐私 → 剪贴板
- Windows：确保没有安全软件阻止

**原因 2：内容过大**
- 公众号有内容长度限制（通常 1-2MB）
- 尝试分批复制长文档

**原因 3：微信编辑器限制**
- 某些样式在微信编辑器中不兼容
- 建议使用默认主题测试

---

### Q2.2：预览显示正常，但复制后格式错乱？

**A：** 这通常是微信编辑器的问题。请尝试：

1. **使用「发草稿」功能（订阅）**
   - 直接发送到草稿箱
   - 避免手动复制

2. **调整复制方式**
   - 先复制，再粘贴到记事本
   - 再从记事本复制到微信编辑器

3. **使用兼容模式**
   - 在插件设置中开启「兼容模式」
   - 简化样式输出

---

### Q2.3：如何一次性复制多篇笔记？

**A：** 目前不支持同时复制多篇笔记。可用解决方案：

**方案 1：创建汇总笔记**
```markdown
# 文章汇总

## 第一篇
[[笔记1]]

## 第二篇
[[笔记2]]
```

**方案 2：使用模板**
1. 创建包含多篇笔记链接的模板
2. 动态生成汇总文档

---

### Q2.4：复制速度慢，如何优化？

**A：** 优化建议：

1. **减少图片数量**
   - 单篇文章不超过 10 张图片
   - 使用压缩后的图片

2. **简化样式**
   - 避免过深的 CSS 选择器
   - 减少复杂动画

3. **分批处理**
   - 长文档拆分为多篇
   - 多次复制

4. **关闭实时预览**
   - 在插件设置中关闭「实时预览」

---

### Q2.5：可以导出为 PDF 或 Word 吗？

**A：** 目前插件不支持直接导出。替代方案：

1. **导出为 HTML**
   - 复制内容到浏览器
   - 打印 → 保存为 PDF

2. **使用第三方工具**
   - Markdown to PDF 转换器
   - Pandoc 工具链

3. **微信文章链接**
   - 发送草稿后获取链接
   - 保存链接而非文件

## 3. 功能相关

### Q3.1：代码高亮不显示？

**A：** 请检查：

1. **代码块语法**
```markdown
正确：
```javascript
const x = 1;
`` `

错误：
```
javascript
const x = 1;
```
```

2. **语言名称**
   - 使用完整的语言名称：`javascript` 而不是 `js`
   - 支持：`python`, `java`, `cpp`, `html`, `css` 等

3. **主题下载**
   - 确认代码高亮主题已下载
   - 在插件设置中检查

4. **行数显示**
   - 检查插件设置中的「显示行数」选项

---

### Q3.2：数学公式不渲染？

**A：** 解决方案：

1. **检查语法**
```markdown
LaTeX：
行内：$E=mc^2$
行间：$$\int_a^b f(x)dx$$

AsciiMath：
行内：`E=mc^2`
行间：``sqrt(a^2+b^2)``
```

2. **设置默认语法**
   - 插件设置 → 数学公式语法
   - 选择 LaTeX 或 AsciiMath

3. **实时预览**
   - LaTeX：安装 MathJax 或 KaTeX 插件
   - AsciiMath：安装 AsciiMath 插件

4. **代码块语法**
```markdown
明确指定语法：
``` latex
c=\pm\sqrt{a^2+b^2}
`` `

``` am
c=+-sqrt(a^2+b^2)
`` `
```

---

### Q3.3：图片上传失败？

**A：** 可能原因：

1. **图片问题**
   - 格式：仅支持 JPG, PNG, GIF
   - 大小：不超过 10MB
   - 路径：不能有中文或特殊字符

2. **网络问题**
   - 检查网络连接
   - 尝试切换网络

3. **账号问题**
   - 确认已购买订阅
   - 验证微信账号权限

4. **错误代码**
   - 45009：图片过大
   - 40001：账号无效
   - 40003：图片格式不支持

---

### Q3.4：文件嵌入不显示？

**A：** 请确认：

1. **文件路径正确**
```markdown
正确：
[[笔记名称]]
[[笔记名称#章节标题]]

错误：
[[/路径/到/笔记]]
```

2. **文件存在**
   - 确认引用的文件确实存在
   - 检查文件名是否完全匹配

3. **权限问题**
   - 确认 Obsidian 有读取该文件的权限

4. **插件设置**
   - 检查插件是否启用了「文件嵌入」功能

---

### Q3.5：Callout 不显示样式？

**A：** 解决方案：

1. **语法正确**
```markdown
正确：
> [!note] 标题
> 内容

错误：
> 标题
> 内容
```

2. **支持的类型**
   - note, tip, warning, success, question, failure, example, quote

3. **主题问题**
   - 某些主题可能不支持 Callout
   - 尝试更换主题或使用自定义 CSS

## 4. 样式相关

1. 打开 NoteToMP 插件

**方法 3：前言元数据**
```yaml

```

---

### Q4.2：自定义 CSS 不生效？

**A：** 检查清单：

1. **笔记存在**
   - 确认自定义 CSS 笔记确实存在

2. **路径正确**
   - 插件设置中的「自定义 CSS 笔记」字段应填入笔记名称（不含扩展名）

3. **CSS 语法**
```css
正确：
.note-to-mp {
    font-family: Arial;
}

错误：
.note-to-mp
    font-family: Arial;
```

4. **刷新预览**
   - 修改 CSS 后需要刷新预览

5. **缓存问题**
   - 关闭重新打开插件界面

---

### Q4.3：如何创建自定义主题？

**A：** 步骤：

**第一步：创建主题文件**
1. 创建笔记「我的主题」
2. 写入 CSS 代码

**第二步：配置插件**
1. 在「自定义 CSS 笔记」中填入「我的主题」

**第三步：优化样式**
```css
/* 示例：简洁主题 */
.note-to-mp {
    font-family: -apple-system, BlinkMacSystemFont, sans-serif;
    line-height: 1.8;
    color: #333;
    max-width: 800px;
    margin: 0 auto;
    padding: 20px;
}

.note-to-mp h1 {
    font-size: 2em;
    color: #2c3e50;
    border-bottom: 3px solid #3498db;
    padding-bottom: 10px;
}

.note-to-mp h2 {
    color: #34495e;
    margin-top: 1.5em;
}

.note-to-mp p {
    margin: 1em 0;
    text-align: justify;
}

.note-to-mp blockquote {
    border-left: 4px solid #95a5a6;
    padding-left: 15px;
    color: #7f8c8d;
    font-style: italic;
}
```

```css
.note-to-mp {

.note-to-mp h1 {

}

.note-to-mp h2 {
    font-size: 1.5em;  /* 24px */
}

.note-to-mp h3 {
    font-size: 1.2em;  /* 19px */
}
```

**方案 3：仅正文**
```css
.note-to-mp p {
    font-size: 18px;
}
```

**A：** 卸载步骤：

**完全卸载：**
1. 禁用插件

2. 删除插件文件
   - 删除 `.obsidian/plugins/note-to-mp/` 目录

3. 清理数据（可选）

**保留设置：**
- 只禁用插件，不删除文件
- 重新安装时会恢复设置

**注意：**
- 卸载后无法恢复草稿和设置
- 卸载前请做好备份

## 7. 故障排除

### Q7.1：插件无法打开？

**A：** 解决方案：

1. **重启 Obsidian**
   - 完全退出再打开

2. **检查插件状态**
   - 设置 → 第三方插件
   - 确认插件已启用

3. **查看错误信息**
   - 按 F12 打开控制台
   - 查看是否有错误日志

4. **重新安装**
   - 禁用插件
   - 卸载插件
   - 重新安装

5. **日志文件**
   - 查看 `.obsidian/logs/` 目录
   - 寻找相关错误信息

---

### Q7.2：预览界面空白？

**A：** 故障排除：

**步骤 1：检查笔记**
- 确认笔记不为空
- 检查笔记路径是否正确

**步骤 2：刷新预览**
- 点击插件界面中的「刷新」按钮

**步骤 3：重置视图**
1. 关闭插件窗口
2. 重新打开

**步骤 4：检查自定义 CSS**
- 清空自定义 CSS 笔记
- 测试是否是 CSS 冲突

**步骤 5：查看控制台**
```javascript
// 在浏览器控制台中执行
console.log('NoteToMP Debug');
```

---

### Q7.3：复制后粘贴只有纯文本？

**A：** 可能原因：

**原因 1：剪贴板格式**
- 微信编辑器可能只支持纯文本模式
- 尝试使用「发草稿」功能

**原因 2：权限问题**
- macOS：系统设置 → 隐私与安全 → 剪贴板
- 确保 Obsidian 有剪贴板权限

**原因 3：内容过大**
- 微信编辑器对内容大小有限制
- 尝试分段复制

**原因 4：兼容性**
- 使用「兼容模式」设置
- 简化输出样式

---

### Q7.4：微信图片上传失败？

**A：** 解决步骤：

1. **检查图片**
   - 格式：JPG, PNG, GIF
   - 大小：不超过 10MB
   - 路径：无中文和特殊字符

2. **检查账号**
   - 确认已购买订阅
   - 验证微信账号状态

3. **网络检查**
   - 测试网络连接
   - 尝试更换网络

4. **查看错误码**
   - 45009：图片过大，压缩后重试
   - 40001：账号无效，联系客服
   - 40003：格式不支持，转换格式

5. **分批上传**
   - 减少单次上传图片数量
   - 分批进行

### Q7.5：设置保存失败？

**A：** 解决方案：

1. **检查权限**
   - 确保 Obsidian 有写入权限
   - 检查插件目录权限

2. **磁盘空间**
   - 确保有足够磁盘空间
   - 清理缓存文件

3. **冲突插件**
   - 暂时禁用其他插件
   - 逐个测试

4. **重置设置**
   - 删除设置文件：`.obsidian/data/note-to-mp.json`
   - 重启 Obsidian
   - 重新配置
