# NoteToMP 长图展示功能 ✨

## 📖 功能概述

长图展示是 NoteToMP 的全新功能，专为展示超长内容设计的容器，支持信息图、流程图、长截图等内容的优雅展示。

## 🎯 核心特性

### 1. 垂直滚动浏览
- **平滑滚动**：流畅的滚动体验
- **视窗限制**：最大高度为屏幕的 80%，避免内容过长影响阅读
- **原生滚动条**：使用浏览器原生滚动，性能更优

### 2. 智能适配
- **自适应宽度**：图片宽度自动适应容器
- **宽高比保持**：保持原始图片的宽高比不变形
- **响应式设计**：适配桌面和移动设备

### 3. 视觉设计
- **极客杰尼主题**：与插件主题完美融合
- **渐变背景**：温暖的橙色渐变背景
- **圆角设计**：8px 圆角，现代化视觉风格
- **精致阴影**：subtle 阴影效果，增强层次感

### 4. 用户体验
- **可选标题**：可添加自定义标题说明
- **快速加载**：优化的图片加载机制
- **无障碍访问**：支持键盘导航和屏幕阅读器

## 🎯 最佳实践

### 1. 图片准备
- **格式选择**：推荐使用 JPG 或 PNG
- **尺寸建议**：
  - 宽度：800-1000px
  - 高度：根据内容调整，建议 1000-2000px
- **文件大小**：建议控制在 1MB 以内

### 2. 标题使用
- **简洁明了**：标题应简洁概括图片内容
- **版本信息**：可在标题中加入版本号或日期
- **示例**：
  - ✅ 系统架构设计图 v2.1
  - ✅ 用户旅程地图（2024版）
  - ❌ 这是一个非常非常长且没有意义的标题

### 3. 内容组织
```markdown
# 产品架构文档
```

## 🚀 技术实现

### 核心特性
- **CSS Grid/Flexbox**：现代化布局技术
- **Viewport Units**：使用 vh 单位限制高度
- **Smooth Scrolling**：原生平滑滚动
- **Object-fit**：智能图片缩放

### 性能优化
- **懒加载**：图片按需加载
- **缓存机制**：已渲染的图片会缓存
- **压缩传输**：支持 WebP 等现代格式
